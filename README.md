### Frontend (React)

1.  **Navigate to the frontend directory:**

    ```bash
    cd frontend
    ```
2.  **Install dependencies:**

    ```bash
    npm install
    ```
3.  **Start the development server:**

    ```bash
    npm run dev
    ```

### Backend (Java)

1.  **Navigate to the backend directory:**

    ```bash
    cd finder
    ```

2.  **Run the application using Gradle:**

    ```bash
    ./gradlew bootRun
    ```
